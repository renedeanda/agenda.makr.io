## Overview

**Meeting Agenda Planner** is a beautiful and performant web application for planning and managing meeting agendas. It helps users create and organize agenda items efficiently, making meetings more productive.

## Features

- Create and organize agenda items
- Real-time timer for each agenda item
- Export agenda as PDF
- Dark mode support
- Responsive design
- Persistent storage using localStorage

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/renedeanda/agenda.makr.io.git
   ```

2. Navigate to the project directory:
   ```bash
   cd meeting-agenda-planner
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Technologies Used

- Next.js

- jsPDF

## Customization

You can customize the colors and styles by modifying the `tailwind.config.js` file and the `globals.css` file in the `styles` directory.
